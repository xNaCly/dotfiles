Included Scripts:

Install scripts:

  • install_fonts.sh
    installs fonts specified in ~/.config/fonts
  • install_packages.sh
    installs packages specified in ~/.config/install_scripts/packages.txt
  • install.sh
    calls the scripts above

i3 scripts:

  • rofi.sh
    calls rofi (this is due to i3 having problems with commas)
  • select.sh
    reads the wallpapers in ~/.config/wallpapers pipes them into dmenu and allows the user to select a new wallpaper, after that the script runs pywal and colors.py
  • colors.py
    reads the colors from ~/.cache/wal/colors.json and inserts them into dunst and i3status. It also restarts and reloads i3 & dunst + copies the template theme to rofis config dir
