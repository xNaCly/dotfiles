xnacly's Dotfiles

These Dotfiles include dynamic wallpaper depended colorschemes for i3, i3bar, rofi, dunst, firefox, vscode and the terminal.

Each i3 reload applies a different wallpaper + a colorscheme based on it using pywal to create and set wallpapers. This script includes a wallpaper selector made with dmenu.

Included Software configs:

  • nvim
  • i3
  • i3status
  • fish
  • alacritty
  • dunst
  • rofi

Included Scripts:

Install scripts:

  • install_fonts.sh
    installs fonts specified in ~/.config/fonts
  • install_packages.sh
    installs packages specified in ~/.config/install_scripts/packages.txt
  • install.sh
    calls the scripts above

i3 scripts:

  • rofi.sh
    calls rofi (this is due to i3 having problems with commas)
  • select.sh
    reads the wallpapers in ~/.config/wallpapers pipes them into dmenu and allows the user to select a new wallpaper, after that the script runs pywal and colors.py
  • colors.py
    reads the colors from ~/.cache/wal/colors.json and inserts them into dunst and i3status. It also restarts and reloads i3 & dunst + copies the template theme to rofis config dir
  • keybinds.py
    reads the configured keybinds and flags from ~/.config/i3/config and inserts them into the template file ~/.config/i3/scripts/keybinds.ms. Allows the user to view all keybinds and flags in pdf format.
